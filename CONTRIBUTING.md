## ğŸš€ How to Contribute

### 1. Fork & Clone
1.  Fork the repo on GitHub.
2.  Clone your fork to your local machine:
    ```bash
    git clone https://github.com/YOUR-USERNAME/LinguaReader.git
    cd LinguaReader
    ```

### 2. Development Setup
To run the project locally, you'll need **Node.js** (v18+) and **NPM**.

```bash
# Install dependencies
npm install

# Run the development server (Vite + Electron)
# This will open the app in a new window with Hot Module Replacement enabled.
npm run dev
```

### 3. Create a Branch
Create a new branch for your specific change. Avoid working directly on the `main` branch.

```bash
git checkout -b feature/my-new-feature
# or
git checkout -b fix/issue-description
```

### 4. Make Changes
Implement your feature or bug fix.
*   Keep your changes focused on the specific task.
*   Test your changes locally to ensure the app still runs correctly.

### 5. Commit & Push
Commit your changes with a clear message describing what you did.

```bash
git commit -m "feat: add support for Italian language"
git push origin feature/my-new-feature
```

### 6. Submit a Pull Request
1.  Go to the [LinguaReader repository](https://github.com/yusufdinc974/LinguaReader) on GitHub.
2.  You should see a prompt to create a Pull Request from your branch.
3.  Click **"New Pull Request"**.
4.  Describe your changes in detail. What did you fix? How did you test it?
5.  Submit!

*   Your operating system (Linux, Windows, macOS).
